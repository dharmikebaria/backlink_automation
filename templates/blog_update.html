Writing Guidelines
  • Title: Write the title on the first line only.
  • Content: Start writing the content from the second line.
  • Hyperlinks: Use format [Text](URL)
    Example: [Softtop](https://www.softtop.tech/)
  • Images: Add on a new line as {image.ext}
    Example: {c1.jpg} (No blank lines before/after)
  • Spacing: No gap required between paragraphs.
